## Usage

As a prerequisite it's assumed you have `npm/yarn` installed and the `gulp-cli`.

1. Clone the repo

        git clone https://github.com/jh3y/parcel-react-boilerplate.git

2. Install dependencies

        npm install / yarn

3. Start

        make develop
